== README

usage:
* git clone git@github.com:radiocontroller/crawler_with_sinatra.git

* cd crawler_with_sinatra

* bundle install

* crontab -e

\* \* \* \* \* /bin/bash -l -c 'cd /your_project_path/crawler_with_sinatra && RAILS_ENV=production bundle exec rake crawler:crawl_all --silent'

* rerun 'thin start -e production'
